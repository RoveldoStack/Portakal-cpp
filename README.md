# Portakal

Portakal is a cross-platform, graphics API-agnostic rendering and compute library for C++.

Supported compilers
- [X] MSVC
- [ ] GNU
- [ ] Clang

Supported platforms
- [X] Windows
- [ ] Linux
- [ ] Mac
- [ ] Android
- [ ] IOS

Supported graphics APIs:

- [ ] OpenGL
- [ ] OpenGL-ES
- [ ] Directx 11
- [ ] Directx 12
- [ ] Vulkan
- [ ] Metal
